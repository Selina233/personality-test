# 问卷版本变更对照表

## 版本对比

### v1.0.0 → v2.0.0 (计划中)

#### 行为测试题变更
**v1.0.0 (2题)** → **v2.0.0 (30题)**

新增题目：
- b3 ~ b30: 新增28道行为测试题
- 涵盖：运动、睡眠、阅读、团体活动、工作/学习、社交、娱乐、压力感知等维度

#### BFI-2测试题变更
**v1.0.0 (5题)** → **v2.0.0 (60题)**

新增题目：
- bfi6 ~ bfi60: 新增55道BFI-2题目
- 每个维度从1题扩展到12题
- 维度分布：
  - 外向性 (E): 12题
  - 宜人性 (A): 12题
  - 尽责性 (C): 12题
  - 神经质 (N): 12题
  - 开放性 (O): 12题

#### 变更原因
1. **测试完整性**：v1.0.0仅用于快速测试，数据可靠性有限
2. **学术规范**：完整版BFI-2需要60题才能计算15个子维度
3. **行为丰富度**：30道行为题能更全面地捕捉用户的生活模式

## 版本数据兼容性

### 数据结构
所有版本的用户数据都包含 `questionnaireVersion` 字段，用于追溯：

```json
{
  "userInfo": {
    "questionnaireVersion": "v1.0.0",
    ...
  },
  "behaviorAnswers": {...},
  "bfi2Answers": {...},
  "bfi2Scores": {...}
}
```

### 版本识别规则
- v1.x.x: 测试版本（7题）
- v2.x.x: 完整版本（90题）
- 数据分析时根据版本号进行分组处理
